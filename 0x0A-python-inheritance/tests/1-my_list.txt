Prints the list
===============
class inherits from list

given values are of type int

    >>> MyList = __import__('1-my_list').MyList

    basic list

    >>> my_list = MyList([1, 4, 2, 3, 5])
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

    1 element

    >>> my_list = MyList([9])
    >>> my_list.print_sorted()
    [9]

    1st element larger

    >>> my_list = MyList([9, 4, 2, 3, 5])
    >>> my_list.print_sorted()
    [2, 3, 4, 5, 9]

    No arguments given

    >>> my_list = MyList()
    >>> my_list.print_sorted()
    []

    empty list

    >>> my_list = MyList([])
    >>> my_list.print_sorted()
    []
