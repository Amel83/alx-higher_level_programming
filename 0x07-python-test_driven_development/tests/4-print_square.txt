# 4-print_square.txt

============================
How to Use 4-print_square.py
============================

A square-printing function.

Usage
=====

# character is used to print the quares.

::

    >>> print_square = __import__('4-print_square').print_square
    >>> print_square(1)
    #

::

    >>> print_square(4)
    ####
    ####
    ####
    ####

::

    >>> print_square(7)
    ##########
    ##########
    ##########
    ##########
    ##########
    ##########
    ##########

when size = 0, the function prints nothing.

::

    >>> print_square(0)

size must be an integer. Otherwise, a TypeError is raised.

::

    >>> print_square("olla")
    Traceback (most recent call last):
    TypeError: size must be an integer

::

    >>> print_square(10.3)
    Traceback (most recent call last):
    TypeError: size must be an integer

::

    >>> print_square(None)
    Traceback (most recent call last):
    TypeError: size must be an integer

When size < 0, a ValueError is raised.

::

    >>> print_square(-7)
    Traceback (most recent call last):
    ValueError: size must be >= 0

Note that type-checking occurs before value-checking.

::

    >>> print_square(-10.5)
    Traceback (most recent call last):
    TypeError: size must be an integer

for no arg given

::

    >>> print_square()
    Traceback (most recent call last):
    TypeError: print_square() missing 1 required positional argument: 'size'
